Detailed Description

Pattern specific configuration structure.

This extends the fiftyoneDegreesConfigDeviceDetection structure by adding collection configurations and options for the allowable closest signatures and the User-Agent cache.

Examples
Pattern/FindProfiles.c, Pattern/GettingStarted.c, Pattern/MatchForDeviceId.c, Pattern/MatchMetrics.c, Pattern/OfflineProcessing.c, Pattern/ReloadFromFile.c, Pattern/ReloadFromMemory.c, and Pattern/StronglyTyped.c.

int32_t fiftyoneDegreesConfigPattern::difference

The maximum difference to allow when matching.

If the difference is exceeded, the result is considered invalid and values will not be returned. By default this is 10.
